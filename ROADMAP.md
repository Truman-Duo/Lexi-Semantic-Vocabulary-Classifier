# Lexi 产品路线图

## v0.0.2（已完成）— 多格式导出 + GUI + 置信度分类

- [x] GUI 图形界面（tkinter）
- [x] CSV / HTML / Anki 三种导出格式
- [x] 派生词模糊匹配（28种后缀模式）
- [x] 置信度评分体系（dictionary=1.0 / fuzzy=0.8 / suffix=0.6 / pos_fallback=0.4）
- [x] CEFR 等级映射（A1-C2）
- [x] 用户自定义覆盖
- [x] 惰性导入 + 进度回调

---

## P0（已完成）— AI 短文生成

**目标**：用户选择 10-50 个目标词汇，AI 生成自然融入这些词的英文短文。

- [x] API 配置管理（`~/.lexi/config.json`，OpenAI 兼容接口）
- [x] 自动选词策略（balanced / top_frequency / random / stratified）
- [x] Prompt 模板（en/zh 双语，支持 CEFR 上下文）
- [x] Markdown 输出（目标词高亮 + 词汇使用状态表）
- [x] CLI 子命令（`python cli.py story` / `python cli.py config`）
- [x] GUI 集成（API 设置对话框 + 短文生成面板）
- [x] 风格模板系统（参考文本 → 风格教练，生成时可选风格）
- [x] 架构解耦（LexiController 统一入口）
- [x] 测试套件（70 个 pytest 单元测试）

---

## P1 — 学习状态追踪 + 间隔复习

**目标**：从"一次性分类工具"变成"持续学习工具"。

- [ ] 学习状态记录（`~/.lexi/learned.json`：每个词的学习次数、最后复习时间、掌握程度）
- [ ] 在分类树中标注"已掌握 / 学习中 / 未学"状态
- [ ] 艾宾浩斯间隔复习提醒（1天 / 3天 / 7天 / 30天）
- [ ] 每日复习词表生成
- [ ] 各分类掌握率统计

---

## P2 — AI 练习生成

**目标**：短文读懂不等于词掌握，需要主动回忆型练习。

- [ ] 完形填空生成（目标词挖空，提供词表选择）
- [ ] 选择题生成（近义词辨析、语境选词）
- [ ] 造句练习 + AI 批改反馈
- [ ] 英英释义匹配题

---

## P3 — 分阶段学习计划

**目标**：帮助用户规划长期学习路径。

- [ ] 设定 CEFR 目标等级（如 B2）
- [ ] 自动筛选目标等级的未掌握词汇
- [ ] 生成每日学习计划（每日 N 词）
- [ ] 每学完一组自动生成短文巩固
- [ ] 学习进度可视化

---
